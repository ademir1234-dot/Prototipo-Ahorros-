# Prototipo-Ahorros-VozCoop
Prototipo ejecutable 

VozCoop es una aplicación diseñada para facilitar la gestión financiera y el registro de transacciones en grupos de ahorro comunales. El proyecto tiene un fuerte enfoque en la inclusión y la accesibilidad, eliminando barreras tecnológicas para usuarios que viven en zonas rurales sin conexión a internet o que tienen dificultades para leer y escribir.

Producto Mínimo Viable (MVP)

La fase inicial del proyecto se centra en resolver dos necesidades críticas de alto valor, seleccionadas como prioritarias para nuestro MVP:

*  Funcionamiento Offline:* La aplicación funciona de manera completamente local. Los usarios pueden registrar sus transacciones mediante notas de voz directamente desde su comunidad en cualquier momento, sin depender de una conexión a internet.
*  Reporte anual automatizado:* Los administradores del grupo de ahorro pueden generar y visualizar un resumen consolidado de todos los socios al final del año. Esto elimina la necesidad de revisar cuadernos manualmente y agiliza la toma de decisiones para la distribución de ganancias.

##  Roadmap (Próximas características)

A medida que el proyecto crezca, se integrarán las siguientes funcionalidades basadas en nuestro *Backlog* de "Historias de Usuario":

* **Registro de transacciones por voz:** Botones accesibles para que los socios dicten sus montos de ahorro, préstamos recibidos y pagos de intereses, clasificándolos automáticamente (ej. préstamos en azul, intereses en naranja).
* **Historial visual intuitivo:** Pantallas de transacciones pasadas basadas en colores e íconos, diseñadas para entender la situación financiera sin necesidad de leer texto.
* **Sincronización inteligente:** El sistema detectará automáticamente cuando haya señal de internet y sincronizará los datos locales con la base de datos del administrador y guarda la información en la nube.
* **Confirmación de voz:** Un sistema de seguridad que repetirá lo que entendió el micrófono para que el usuario confirme antes de guardar el registro, evitando errores.
* **Identificación por voz:** Reconocimiento del usuario mediante su nombre hablado para vincular el registro a su cuenta sin intervención de terceros.
